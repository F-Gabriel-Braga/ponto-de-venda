# Ponto de Venda

## Descrição
Ponto de Venda é um aplicação que se baseia em um sistema de gerenciamento comercial.

## Tecnlogias
* Java
* JavaFX
* JavaFX CSS
* MySQL
* JDBC
* JFoenix
